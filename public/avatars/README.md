# Team Avatar Images

Place your team member avatar images in this folder.

## Naming Convention

- `dan.png` - Dan (Dexter Memo) avatar
- `david.png` - David avatar
- `nano.png` - Nano avatar

## Image Specifications

- **Recommended size**: 200x200 pixels
- **Format**: PNG with transparency or JPG
- **Aspect ratio**: 1:1 (square)
- **File size**: Under 100KB recommended

## Notes

- If no image is found, the avatar will show the initials with a gradient background
- Dan: Blue gradient (DM)
- David: Green gradient (DV)
- Nano: Purple gradient (NN)
