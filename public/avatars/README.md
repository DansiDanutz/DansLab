# Team Avatar Images

Place your team member avatar images in this folder.

## Naming Convention

- `dan.png` - Dan (Owner / Human Boss) avatar
- `david.png` - David (Fleet Orchestrator) avatar
- `dexter.png` - Dexter (General Manager) avatar
- `nano.png` - Nano (Agent Creator) avatar
- `sienna.png` - Sienna (Crypto Specialist) avatar
- `memo.png` - Memo (Project Manager) avatar

## Image Specifications

- **Recommended size**: 200x200 pixels
- **Format**: PNG with transparency or JPG
- **Aspect ratio**: 1:1 (square)
- **File size**: Under 100KB recommended

## Notes

- If no image is found, the avatar will show the agent's initials with a colored background
- Dan: White/Light gradient (DN)
- David: Green gradient (DV)
- Dexter: Blue gradient (DX)
- Nano: Purple gradient (NN)
- Sienna: Pink gradient (SI)
- Memo: Orange gradient (MM)
